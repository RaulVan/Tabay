# Chrome 网上应用店商品描述

## 简短描述
Tabay - 强大的标签页管理工具，帮助你更好地组织浏览器标签。

## 详细描述
Tabay 是一款简洁而强大的标签页管理工具，专为需要同时处理大量标签页的用户设计。它能帮助你轻松组织和管理 Chrome 浏览器中的标签页，提高工作效率。

### 主要功能
✨ 一键保存
- 一键保存所有标签页到新分组
- 智能过滤不支持的页面类型
- 自动同步到 Chrome 账号

📂 分组管理
- 自定义标签组名称
- 按时间自动分类（今天/昨天/本周/更早）
- 拖拽标签在不同组之间移动
- 一键打开组内所有标签

🔍 快速搜索
- 实时搜索标签页
- 支持标题和网址搜索
- 智能过滤空标签组

💾 数据同步
- 自动同步到 Chrome 账号
- 支持导出数据备份
- 支持导入数据恢复
- 显示同步状态和进度

🎨 界面设计
- 简洁现代的界面
- 支持浅色/深色主题
- 标签组展开/折叠
- 拖拽操作视觉反馈

### 使用提示
1. 点击工具栏图标打开管理界面
2. 使用"保存全部"快速保存当前窗口所有标签
3. 输入关键词实时搜索已保存标签
4. 拖拽标签调整分组
5. 点击标签直接打开
