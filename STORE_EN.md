# Chrome Web Store Product Description

## Short Description
Tabay - A powerful tab management tool that helps you better organize your browser tabs.

## Detailed Description
Tabay is a sleek and powerful tab management tool designed for users who work with multiple tabs. It helps you effortlessly organize and manage tabs in Chrome browser, boosting your productivity.

### Key Features
✨ One-Click Save
- Save all tabs to a new group with one click
- Smart filtering of unsupported page types
- Auto-sync with Chrome account

📂 Group Management
- Custom group names
- Automatic time-based categorization (Today/Yesterday/This Week/Earlier)
- Drag and drop tabs between groups
- Open all tabs in a group with one click

🔍 Quick Search
- Real-time tab search
- Search by title and URL
- Smart filtering of empty groups

💾 Data Sync
- Automatic sync with Chrome account
- Export data backup
- Import data recovery
- Sync status and progress display

🎨 Interface Design
- Clean and modern interface
- Light/Dark theme support
- Expandable/collapsible groups
- Visual feedback for drag operations

### Usage Tips
1. Click the toolbar icon to open management interface
2. Use "Save All" to quickly save all tabs in current window
3. Enter keywords to search saved tabs in real-time
4. Drag tabs to rearrange groups
5. Click tabs to open directly
